Sistema completo de produtividade e transformação pessoal.

## 🚀 Funcionalidades

- 🚫 **Combate a Vícios** - Rastreie e supere dependências
- 🔥 **Hábitos Atômicos** - Crie e mantenha hábitos
- 📚 **Técnicas de Estudo** - Pomodoro, Active Recall, Spaced Repetition
- ⏰ **Timer Pomodoro** - Foco profundo com pausas ativas
- ✅ **GTD** - Getting Things Done
- ✨ **Sistema M** - Gerencie múltiplos interesses
- 🎮 **Gamificação** - XP, níveis, streaks e conquistas
- 📊 **Analytics** - Entenda seus padrões

## 🛠️ Tecnologias

- **Frontend**: Next.js 15 + React + TypeScript
- **Estilização**: Tailwind CSS
- **Estado**: Zustand
- **Banco**: IndexedDB (offline-first)
- **PWA**: Instalável como app

## 💾 Instalação

```bash
# Clone
git clone <seu-repo>

# Instale
npm install

# Rode
npm run dev
```

Acesse: **http://localhost:3000**

## 📱 Instalar como App

### Android/iOS:
1. Abra no navegador
2. **Chrome**: Menu → "Adicionar à tela inicial"
3. **Safari**: Compartilhar → "Adicionar à Tela de Início"
