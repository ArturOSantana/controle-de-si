# Ícones do Menu

Esta pasta contém as imagens personalizadas para os ícones do menu.

## Imagens Necessárias (formato PNG ou SVG, tamanho recomendado: 64x64px)

- `home.png` - Ícone de Início
- `habits.png` - Ícone de Hábitos
- `pomodoro.png` - Ícone de Pomodoro
- `tasks.png` - Ícone de Tarefas
- `addictions.png` - Ícone de Vícios
- `study.png` - Ícone de Estudos
- `timeblock.png` - Ícone de Agenda
- `journal.png` - Ícone de Diário
- `analytics.png` - Ícone de Análises
- `reports.png` - Ícone de Relatórios
- `achievements.png` - Ícone de Conquistas
- `rewards.png` - Ícone de Recompensas

## Como usar

1. Adicione as imagens nesta pasta com os nomes listados acima
2. Use o botão de alternância no canto superior direito para trocar entre ícones do sistema e imagens personalizadas
3. As imagens serão carregadas automaticamente quando o modo de imagens estiver ativo
